# Test for Wild Code Scholl

Tools used
--------------------

- Backend: Node.js, Express.js, Mongoose
- Database: MongoDB
- Frontend: React, hooks, axios

How to run app locally
--------------------

- Frontend
  * `cd front/`
  * `yarn`
  * `yarn start`

- Backend
  * `cd back`
  * `yarn`
  * `nodemon server.js` (to use nodemon)
